# Onjava

This repository contains two main projects demonstrating Java and React integration.

## Projects

### Jar (Java Spring Boot Application)
A Spring Boot application with REST endpoints, security configuration, and a message repository.

- **Location**: `jar/`
- **Main Class**: `com.jar.jar.JarApplication`
- **Features**:
  - REST API with HelloController
  - User authentication and security
  - Message model and repository

### My React App
A React application that likely interacts with the Java backend.

- **Location**: `my-react-app/`
- **Features**:
  - React components
  - API service for communication

## Getting Started

### Prerequisites
- Java 11+ for the Jar project
- Node.js and npm for the React app

### Running the Jar Application
1. Navigate to `jar/`
2. Run `./mvnw spring-boot:run`

### Running the React App
1. Navigate to `my-react-app/`
2. Run `npm install`
3. Run `npm start`
